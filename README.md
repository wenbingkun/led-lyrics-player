# LED Lyrics Player | LED 演出歌词显示器

一个专为舞台演出或视频录制设计的 Web 歌词播放器，具有独特的 "LED 风格" 视觉效果。支持音频同步、多种主题切换、以及便捷的文件管理。

## ✨ 特性

-   **视觉震撼**: 超大 LED 风格字体，专为大屏幕和演出场景优化。
-   **音画同步**: 支持加载本地音频（MP3/WAV/FLAC 等）与 LRC 歌词文件自动匹配并同步播放。
-   **智能匹配**: 内置智能模糊匹配算法，能够自动关联歌词与同名（或相似名）音频文件。
-   **操控便捷**: 支持拖拽上传文件、键盘快捷键、数字键快速选歌。
-   **高度自定义**:
    -   四种内置主题：经典白、流金岁月、赛博蓝、炫彩霓虹。
    -   可调节播放速度（0.5x - 2.0x）。
    -   可调节字体缩放和背景图片。
-   **性能卓越**: 基于 `requestAnimationFrame` 驱动，动画平滑，低内存占用。
-   **离线可用**: 纯前端应用，无需服务器后端，数据持久化存储在本地浏览器。

## 🚀 快速开始

由于这是一个静态 Web 应用，你可以直接在浏览器中打开 `index.html`，或者使用任何静态文件服务器运行。

### 使用 Python (推荐)
```bash
python3 -m http.server 8000
```
然后在浏览器访问 `http://localhost:8000`。

## 键盘快捷键 ⌨️

| 按键 | 功能 |
| :--- | :--- |
| `空格` | 播放 / 暂停 |
| `←` / `→` | 后退 / 快进 5 秒 |
| `↑` / `↓` | 切换上一首 / 下一首 |
| `P` / `N` | 上一曲 / 下一曲 |
| `Ctrl + ←` / `→` | 跳转上一句 / 下一句歌词 |
| `1 - 9` | 快速切换到对应的歌曲 |
| `F` | 聚焦搜索框 |
| `T` | 切换视觉主题 |
| `C` | 切换歌曲模式 (同步/音频/歌词) |
| `[` / `]` | 微调音频同步偏移 (±0.1s) |
| `Esc` | 切换全屏模式 |

## 📁 文件管理

1.  **加载歌词**: 拖放 `.lrc` 或 `.txt` 文件到窗口，或在控制面板选择文件夹。
2.  **加载音频**: 拖放音乐文件，播放器会根据歌词文件名自动尝试匹配。
3.  **模式选择**:
    -   `同步模式`: 音频与歌词联动。
    -   `纯歌词模式`: 手动或定时滚动歌词。
    -   `纯音频模式`: 仅播放音频。

## 🛠️ 项目架构

-   **前端**: 原生 HTML5, CSS3 (Flexbox/Grid), ES6+ JavaScript (ES Modules)。
-   **核心逻辑**: `js/player.js` 中的 `LEDLyricsPlayer` 类负责状态管理和渲染调度。
-   **样式**: `css/style.css` 包含所有主题定义和液态玻璃拟态 UI。
-   **测试**: `tests/` 目录下包含完整的自动化单元测试。

## 🧪 开发与测试
